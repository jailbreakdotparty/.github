# jailbreak.party
**we make iOS apps, tools, and scripts.**

## Projects
| Project Name | Description | Supported Software |
| - | - | - |
| [dirtyZero](https://github.com/jailbreakdotparty/dirtyZero) | A simple customization toolbox, utilizing [CVE-2025-24203](<https://project-zero.issues.chromium.org/issues/391518636>). | iOS 16.0 - iOS 18.4 |
| [PancakeStore](https://github.com/jailbreakdotparty/PancakeStore) | A jailed app store app downgrader, based off of MuffinStoreJailed. | iOS 16.4+ |
| [Octopus]() | Coming soon... | iOS 17.0+ |
| [Lithium]() | Coming soon... | iOS 16.0+ |
| [Bridge](https://github.com/jailbreakdotparty/Bridge) | A jailed internal app opener and extractor. | iOS 16.0 - iOS 26.0.1 |
| [Accessible](https://github.com/jailbreakdotparty/Accessible) | A jailed filesystem utility that utilizes the Shortcuts application. | iOS 16.0+ |
| [BlacklistBeGone](https://github.com/jailbreakdotparty/BlacklistBeGone) | Enterprise app blacklist remover using SparseRestore. | iOS 16.0 - iOS 18.0.1 |

Note that these tools also work on iPadOS, though YMMV.
